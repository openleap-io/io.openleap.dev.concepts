# Glossary Term: Bin Location

> **Template:** [Glossary Term](../../../../concepts/artifacts/product/glossary-term.md)

---

## Fields

| Field | Value |
|-------|-------|
| id | bin-location |
| term | Bin Location |
| definition | A specific, addressable physical storage position within the warehouse, identified by a combination of aisle, rack, level, and position. Each bin has a defined capacity and a type (standard, cold, hazmat) that constrains which materials can be stored there. |
| aliases | Bin, Storage Bin, Slot, Lagerplatz |
| functionalAreaRef | storage |

---

## Examples

- "Bin A-03-2-1 means Aisle A, Rack 03, Level 2, Position 1."
- "The cold storage bins are all in Aisle F."
