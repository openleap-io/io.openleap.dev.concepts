# Glossary Term: Lot / Batch

> **Template:** [Glossary Term](../../../../artifacts/product/glossary-term.md)

---

## Fields

| Field | Value |
|-------|-------|
| id | lot-batch |
| term | Lot / Batch |
| definition | A production or delivery lot identifier assigned to a quantity of material. Lots enable traceability (knowing where material came from and when it was received) and are the basis for FIFO picking of perishable goods. |
| aliases | Lot, Batch, Charge, Los |
| functionalAreaRef | storage |

---

## Examples

- "Lot L-2026-0100 contains 500 pcs of Material M-001, received on 2026-03-01."
- "The FIFO rule requires picking from Lot L-2026-0100 before Lot L-2026-0200 because it was received earlier."
